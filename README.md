## How to run the application

1. Clone the project

```bash
  git clone https://github.com/vinicbarros/recime-recipes.git
```

2. Go to the project folder

```bash
  cd recime-recipes
```

3. Install dependencies

```bash
  npm install
```

4. Run the application :)

```bash
  npm run dev
```

5. Access by
```bash
  http://localhost:3000/
```
